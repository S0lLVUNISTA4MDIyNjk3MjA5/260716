目的から操作を開始する

1. Excelを読み込む

シートと変換範囲を選択します。

2. 方眼紙を整形する

結合展開・空白補完・UNKNOWN付与で表を整えます。

3. 表を確認・編集する

検索・一括編集・行レビューで内容を確認します。

4. 様式・タグを設定する

統制語彙・別名・自動付与を設定します。

5. 検証して出力する

未確認・未登録タグを解消して出力します。
