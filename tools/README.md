# Tools

Standalone single-file HTML tools for requirements ⇔ design traceability work.

- `spec_to_json_conversion_tool_v1.18.html`: 仕様書PDF→JSON変換・確認ツール v1.18（取扱説明書準拠UI版）。PDF仕様書から照合用JSONを作成し、様式登録・版間比較・タグ付けを行う。v1.16で確認状態の色分け表示を追加、v1.17でツールバー初期化不具合（作業中JSON・確認済みJSON・検証結果・文書版情報・一括確認・元に戻す・作業復旧ボタンが消える）を修正、v1.18で同梱の様式プロファイル仕様書・生成AI用作成プロンプトをより詳細な版へ更新（ツール内「AIで様式作成」から参照可能）。
- `excel_to_json_conversion_tool_v2.0.8.html`: Excel → JSON変換・確認ツール v2.0.8。Excelの設計検証項目一覧をJSONに変換し、様式定義・タグ辞書に基づき検証する。v2.0.2で確認状態の色分け表示を追加、v2.0.3で「様式・タグ」メニューに「AIで様式作成」を追加してプロファイル仕様書・生成AI用プロンプトを参照可能にし、モーダルの無地ボタンが見えなくなる不具合を修正、v2.0.4でヘッダー・工程バー・「目的から操作を開始する」カードのUI構成を仕様書PDF→JSONツールに近づけ、v2.0.5で配色・タイポグラフィ・ボタン/バッジ/カード/モーダルのデザイントークンを仕様書PDF→JSONツールと統一、v2.0.6で手動タグ付与・変更時に未確認行を自動的に確認済みへ変更するよう修正、v2.0.7で「使い方」を仕様書PDF→JSONツール同様のナビ付きガイド（6項目）へ拡充、v2.0.8で列の値を照合タグへ一括反映する機能と、検索・フィルタ結果を一括確認済みにする機能を追加。
- `json_ab_trace_matching_tool_v12.1.14.html`: JSON A/Bトレース照合ツール v12.1.14。上記2ツールで生成したJSON（要求側A／設計側B）を照合し、ナレッジグラフ・トレースマトリクスを生成する。v12.1.4で入力差替え時の照合キー再検証・詳細表全件フィルタ・重複ID分離等を改善、v12.1.6で詳細テーブル/トレースマトリクス/ナレッジグラフの配色をJSON A（青）・JSON B（橙）・照合情報（緑）で統一し、組み込みプロファイルを汎用の1件に整理、v12.1.7でJSON A表示名・ナレッジグラフの要求側ノードラベルが「[object Object],[object Object]」になる不具合（配列・オブジェクト値をフィールド選定ロジックが誤って文字列化していたため）と、v12.1.6で取り残されていた版数表記（内蔵ヘルプ・生成コード・RO-Crate/ReqIF/Excel/学習データの出力メタデータ）を修正、v12.1.8で詳細テーブルの列過多・行の縦伸びと、ナレッジグラフのノードラベル判読性・選択ノード詳細の生JSON表示を改善（元データ由来の自動抽出列・重複列を既定非表示化しトグルで復元可能に、長文セルを既定3行折り畳み＋クリックで全文表示に、ノードラベルをtrace_title等の短い項目優先に、選択ノード詳細を種別・本文・出典・タグ等の要約表示にしトグルで生JSON表示に切替可能に）、v12.1.9で詳細テーブルの照合情報領域からレビュー内訳系の列（自動照合件数・有効照合件数・人手確定件数・要確認件数・除外件数・除外一覧・レビュー判定内訳・レビューコメント・共有タグ・判定理由）を既定で非表示にし、照合件数・信頼度・照合キー・照合根拠が一目でわかる構成に整理（「レビュー関連の列も表示」トグルで復元可能）、v12.1.10で詳細テーブル（A基準・B基準とも）に「最大信頼度の照合先」列を追加し、照合方式を出さずに最大信頼度がどのA-B対応の結果かを集計ビューのまま直接読み取れるようにした、v12.1.11で1件に複数件が対応する列（表示名一覧・キー一覧・照合根拠）を「/」区切り1行表示から改行＋「1. 」形式の番号付き複数行表示に変更（1件のみの場合は番号なし）、v12.1.12で詳細テーブルの列順を「表示名・照合件数・表示名一覧・最大信頼度・最大信頼度の照合先・照合根拠」に整理し、照合JSON Bキー一覧とB基準のJSON Bキー・補助表示値を既定非表示にした（「元データの詳細列も表示」で復元可能）、v12.1.13で外部AIレビュー指摘を反映し、詳細テーブルのExcel出力で絞り込み結果が0件のときに全件出力されてしまう不具合（空のSetを「フィルタなし」と誤判定）と、JSON B基準・概要グラフでフィルタ適用後に関係線が常に非表示になる不具合（概要グラフのエッジがA→B固定で標準グラフのB基準時source/target入替規約と不整合だった）を修正し、ダウンロードファイル名に残っていた旧版数のアンダースコア表記（V12_1_2/V12_1_3）をV12.1.13へ統一した、v12.1.14で「詳細テーブル」を「照合結果一覧」に改称（タブ名・見出し・列凡例・Excel出力のシート名/ファイル名・ステータス文言まで統一）し、ナレッジグラフのノード色・グループ枠色を照合結果一覧／トレースマトリクスと同じ配色（JSON A=青、JSON B=橙）に統一した（関係線の信頼度による緑/黄/赤の色分けは弱い照合を見つけやすくする指標として維持）。
